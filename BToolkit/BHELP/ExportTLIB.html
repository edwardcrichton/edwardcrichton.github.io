ExportTLIB

`Create TLIB Construct' is available under the TopBar Construct menu and allows a developer to export a (sub) development - a MACHINE for which an IMPLEMENTATION is written and a binary code module exists.

For a full description see Team Library.

The team library directory is set through the Teamlib setting of the Options.
