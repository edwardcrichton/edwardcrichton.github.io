Change Directory

This mechanism allows for a change of development directory without leaving the BToolkit. It is equivalent to:
  1. Pressing the Quit button
  2. Changing directories
  3. Running the BToolkit in the new development directory
It is accessed from the TopBar->Utils menu and provides the selection box:

See figure.

OK selects the Current Selection, as does <CR> at the end of the Current Selection text.

The selection may be changed by either clicking on one of the radio-buttons, or by typing the directory name (full path, or relative to the current development directory) directly into the Selection, and then selected by either OK or <CR>.

Filter changes the directories displayed to those of the Current Selection.

"Double-clicking" on one of the radio buttons has the same effect as clicking the radio-button and then clicking on Filter.

Note that directories without write permission are not displayed.

If the selected directory is not a current BToolkit development directory, confirmation is sought on the creation of the new development.
