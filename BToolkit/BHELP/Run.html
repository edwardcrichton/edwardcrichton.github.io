Run/Execute (exe)

The Execute command is invoked from the Translators Environment and is applicable to implementations which are currently linked, or machines for which Motif interfaces are currently generated.

For the former, the tool provides a window in which execution of the code takes place; the latter automatically provides an X/Motif environment.
