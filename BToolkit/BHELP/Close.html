Close All Open Constructs

This option is invoked from the TopBar Constructs pull-down and presents a selection of currently open constructs (an error results if there are none). Unlike similar menus (with the exception of those for Commit and Remake) all constructs are `pre-selected' so that pressing the `OK' button has the effect of selecting all. The option then proceeds as though the `opn -quit-editor' button had been pressed for each selected construct.

It is also possible to commit the changes of an edited construct through the Editor CommitEdits, Commit, Remake, Analyser, POGenerator and Mini-Remake tools; see Top Bar Help for the appropriate tool.
